# US State Suggestion
To run the project

## For db directory:
cd db
npm install
docker-compose up

cd db
node seed.js

## For backend directory:
Make sure installed Golang 1.22+, and do
`export PATH=$PATH:/usr/local/go/bin`  
`go run main.go`

## For frontend directory:
`cd vue-project`  
`npm install`  
`npm run serve`
